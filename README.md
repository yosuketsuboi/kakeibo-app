# 家計簿アプリ

レシートをOCRで読み取り、支出を記録・管理するWebアプリケーションです。

## 機能

- **レシートOCR読み取り** — カメラで撮影したレシートを自動解析し、明細・合計金額を取り込む
- **手動支出入力** — レシートなしで支出を直接登録
- **カテゴリ管理** — 支出をカテゴリに分類して管理
- **支払方法管理** — クレジットカード・電子マネーなど複数の支払方法を登録・管理
- **ダッシュボード** — 月別支出の推移グラフ、支払方法別・カテゴリ別の内訳を表示
- **支出一覧** — カテゴリフィルタ付きの支出一覧（レシート・手動入力の両方に対応）
- **世帯共有** — 招待リンクで家族と世帯を共有

## 技術スタック

| 項目 | 内容 |
|---|---|
| フレームワーク | Next.js 15（App Router） |
| データベース | Supabase（PostgreSQL + RLS） |
| スタイル | Tailwind CSS |
| デプロイ | Vercel |

## 開発環境のセットアップ

```bash
# 依存パッケージのインストール
npm install

# 環境変数の設定
cp .env.example .env.local
# .env.local に Supabase の接続情報を設定

# 開発サーバーの起動
npm run dev
```

`http://localhost:3000` をブラウザで開いてください。

## ビルド

```bash
npm run build
```

## デプロイ

`main` ブランチへの push で Vercel に自動デプロイされます。
